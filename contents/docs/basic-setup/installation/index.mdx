## Installation

<Step>
<StepItem title="Install from PyPI">
The easiest way to install Taizun is from PyPI:

```bash
pip install taizun
```

</StepItem>
<StepItem title="Install from Source">
Alternatively, you can install from the source code:

```bash
git clone https://github.com/t4zn/taizun.git
cd taizun
pip install -e .
```

</StepItem>

<StepItem title="Verifying Installation">
After installation, you can verify it works by importing Taizun in Python:

```python
import taizun
print("Taizun installed successfully!")
```

</StepItem>

<StepItem title="Basic Usage">
Here's a simple example of how to use Taizun:

```python
import taizun as tz

# Text summarization
text = "Your long text here..."
summary = tz.summarize_text(text)
print(summary)

# Sentiment analysis
sentiment = tz.sentiment_analysis("I love this library!")
print(sentiment)
```

</StepItem>
<StepItem title="Documentation Website">
This documentation website is built with Next.js and serves as a reference for all Taizun functions. You can run it locally:

```bash
git clone https://github.com/t4zn/taizun.git
cd taizun
pnpm install
pnpm run dev
```

This will start the local server, and you can open your browser at:

```arduino
http://localhost:3000
```

</StepItem>
</Step>

## Optional

<Step>
<StepItem title="Installing Additional Dependencies">
Some advanced features may require additional dependencies. These will be automatically installed when needed, or you can install them manually:

```bash
pip install taizun[nlp]      # For advanced NLP features
pip install taizun[cv]       # For advanced Computer Vision features
pip install taizun[all]      # For all features
```
</StepItem>
<StepItem title="Updating Taizun">
To update Taizun to the latest version:

```bash
pip install --upgrade taizun
```
</StepItem>
</Step>
